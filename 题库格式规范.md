# 题库导入格式规范

本APP支持导入 Word (.docx) 和 Markdown (.md) 格式的题库文件。为了确保题目能被正确识别，请遵循以下格式规范。

## 通用规则

1. **题目开始**：每道题建议以题型前缀开头（如“单选题：”），后跟题目内容。
2. **选项**：选项需以 `(A)`、`(B)` 或 `A.`、`B.` 等格式开头。
3. **答案**：必须包含“答案：”关键字。
4. **解析**：必须包含“解析：”关键字（可选）。

## 示例格式

### 1. 单选题

```text
单选题：1. Flutter是哪个公司开发的？
选项：
    (A) Apple
    (B) Google
    (C) Facebook
    (D) Microsoft
答案：（B）
解析：Flutter是Google开发的UI框架。
```

### 2. 多选题

```text
多选题：2. 以下属于Flutter特点的是？
选项：
    (A) 跨平台
    (B) 热重载
    (C) 仅支持Web
    (D) 声明式UI
答案：（A）、（B）、（D）
解析：Flutter支持移动端、Web和桌面端，不仅仅是Web。
```

### 3. 判断题

```text
判断题：3. Dart语言支持空安全。
选项：
    (A) 正确
    (B) 错误
答案：（A）
解析：Dart 2.12版本开始支持空安全。
```

### 4. 简答题

```text
简答题：4. 请简述StatefulWidget的生命周期。
答案：createState -> initState -> didChangeDependencies -> build -> didUpdateWidget -> deactivate -> dispose
解析：这是标准的生命周期流程。
```

### 5. 填空题

```text
填空题：5. Flutter使用()语言进行开发。
答案：Dart
解析：Flutter使用Dart语言。
```

或者支持多个空：

```text
填空题：6. Flutter的两个主要构建块是()和()。
答案：Widget|Element
解析：Widget描述配置，Element管理状态。
```

## 注意事项

- 题目与题目之间最好留有一行空行。
- 标点符号支持中文（：）和英文（:）。
- 选项前的缩进不是必须的，但推荐使用以提高可读性。
- 答案中的括号支持中文`（）`和英文`()`。
