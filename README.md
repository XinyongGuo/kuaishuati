# flutter_application_1
刷题软件，可将文档和提示词发给ai，将修改格式后的word/md文档导入，生成刷题题库。
支持单选、多选、判断、填空、简答，暂不支持图片。
增加了ai导入功能，暂时只支持也只测试了DeepSeek的api。只支持安卓与windows。
